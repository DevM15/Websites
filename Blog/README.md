### Run this app locally

```shell
python manage.py runserver
```

### To Create a SuperUser

```shell
python manage.py createsuperuser
```

### Upload a Blog

- Create a superuser.
- Go to localhost:****/admin.
- Login using your username and password.
- Add New Object to blog.
